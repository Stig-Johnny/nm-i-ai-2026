# Main Competition Rules

See [app.ainm.no/rules](https://app.ainm.no/rules) — Main Competition tab.

**Period:** March 19 (18:00 CET) – March 22 (15:00 CET) — 69 hours
**Tasks:** 3 independent challenges (revealed at kickoff)
**Prize pool:** 1,000,000 NOK (400k / 300k / 200k / 100k)
**Scoring:** Each task normalized 0-100, averaged equally (33.33% each)

- AI assistants (Claude, ChatGPT, Copilot) **explicitly permitted**
- Code must be **public MIT license** for prize eligibility
- **Vipps verification** required for prizes
- No collusion or solution sharing between teams
- Teams: 1-4 members, roster locked after first submission
